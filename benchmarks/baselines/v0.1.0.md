# Benchmark Baseline v0.1.0

Recorded baseline for regression comparison.

## Smoke benchmarks (example reference values)

- `BenchmarkRouterStatic`: ~600 ns/op, 11 allocs/op
- `BenchmarkRouterParam`: ~650 ns/op, 11 allocs/op
- `BenchmarkRouterWildcard`: ~760-830 ns/op, 12 allocs/op
- `BenchmarkMiddlewareChain`: ~2600-2800 ns/op, 22 allocs/op
- `BenchmarkCompareStdHTTPStatic`: baseline comparator
- `BenchmarkCompareElgonStatic`: comparator target

Threshold enforcement is configured in:

- `benchmarks/thresholds.tsv`
- `benchmarks/compare_thresholds.tsv`
- `scripts/bench_guard.sh`
